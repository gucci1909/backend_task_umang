## Run Locally

Clone the project

```bash
  git clone https://github.com/gucci1909/backend_task_umang_arora.git
```

Go to the project directory

```bash
  cd backend
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  node dist/index.js
```
